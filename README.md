# classical-chinese-applet

后续小程序想法:
同时面向学生和普罗大众:
1. 内置古文名录，包括《论语》、《孟子》、《大学》、《中庸》、《周易》、《诗经》、《尚书》、《礼记》、《春秋》、《三国志》、《汉书》、《后汉书》、《老子》、《庄子》、《韩非子》、《墨子》、《孙子兵法》、《楚辞》、《全唐诗》、《乐府诗集》、《古文观止》、《世说新语》、《聊斋志异》、《山海经》、《资治通鉴》、《黄帝内经》、《史记》、《鬼谷子》
2. 之于学生，学习后可以应试；之于普罗大众，学习后通读古文

家长端粗浅想法：
1. 可设置阶段性奖励物品、可付费邮寄实物勋章；
2. 获得实物勋章还需要对学生已掌握的词进行家长端的考察，家长设置获得奖章的通过率指标，比如设置90%；
3. 家长端考察难度会更高，开始考察，则随机提取孩子已掌握但未考察过的词。孩子需准确说出意思，家长主动点击回答正确或回答错误。
