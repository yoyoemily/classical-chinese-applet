### 补齐流程

每次补齐一个年级（如"继续七下诗词补齐"），按以下步骤：

1. 确认本年级清单和 ID 分配
2. 切句 + 逐句译文（校验分句数对齐）
3. keyWords 标注（以 8 本打卡型词书为考纲参照）
4. 典故注释（地名/人名/典故，新建 `art_XXX.json`）
5. 创作背景（每篇约 100 字）
6. 编写数据文件 → `scripts/data/poems_gradeXb.json` + `glossaries_gradeXb.json`
7. 编写导入脚本 → `scripts/generate_gradeXb_poems.py`（参照八上模式：JSON 数据文件 + Python 导入脚本分离，避免中文引号与 Python 字符串冲突）
8. 运行 `--apply` 写入知识库 + 导入数据库 + 导入词书
9. 完成后检查清单（见 [[article-adjustment-workflow]]#8），更新本文对应年级状态为 ✅
