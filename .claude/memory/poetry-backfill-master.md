### 补齐流程

每次补齐一个年级（如"继续七下诗词补齐"），按以下步骤：

1. 确认本年级 12 首清单和 ID 分配
2. 切句 + 逐句译文（校验分句数对齐）
3. keyWords 标注（以 8 本打卡型词书为考纲参照）
4. 典故注释（地名/人名/典故，新建 `art_XXX.json`）
5. 创作背景（每篇约 100 字）
6. 写入知识库分文件 → JSON 校验 → 导入数据库
7. 词书 keyWordRefs 同步 → 导入词书
8. 完成后检查清单（见 [[article-adjustment-workflow]]#8），更新本文对应年级状态为 ✅
